# User Card for describe tweets

Цей проект був створений за допомогою
[Create React App](https://github.com/facebook/create-react-app). Для знайомства
і налаштувань додаткових можливостей
[зверніться до документації](https://facebook.github.io/create-react-app/docs/getting-started).

## У проекті використовувалися

1. Node.js. [завантаження і установка](https://nodejs.org/en/).
2. Компаненти-класи.
3. СSS-модулі.
4. Проект деплоїться на GitHub Pages.

## Функціонал картки-користувача

1. Картка користувача при першому завантаженні на сторінку відображає:
 - аватар користувача
 - кількість твітів
 - кількість підписників
 - функціонална кнопка "Підписатися"
2. При натисканні на кнопку "Підписатися" відбувається додавання числа підписника та змінюється колір та стан кнопки на "Підписаний" 
3. При перезавантаженні сторінки стан кнопки та кількосі підписників зберігається. (Дані завантажуються з Локального Сховища користувача)
